**Omni Synapse** is an AI router built in Rust that provides unified management for LLM, speech-to-text, text-to-speech, embeddings, image generation, and MCP tool providers. Configure your API keys in one place and let Synapse handle provider dispatch, model routing, and protocol translation.

## Why Synapse?

Without Synapse, each application manages its own provider keys, API clients, and routing logic. Synapse centralizes this:

- **Single key store** -- configure API keys once, use them from any consumer
- **Unified API** -- OpenAI-compatible and Anthropic-compatible endpoints for all modalities
- **MCP aggregation** -- discover and call tools from multiple MCP servers
- **Provider abstraction** -- switch providers without changing consumer code
- **Intelligent routing** -- automatically select providers and models based on cost, quality, or custom strategies

## Consumers

Synapse is used by:

- **[Beacon](/grid/beacon)** -- routes all LLM, STT, and TTS traffic through Synapse
- **[Omni CLI](/armory/omni-cli)** -- uses Synapse as an LLM provider via `provider = "synapse"`
- **Any OpenAI-compatible client** -- point your base URL to Synapse and authenticate with your `synapse_` API key
