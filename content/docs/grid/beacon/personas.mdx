Personas define how your AI assistant sounds, behaves, and responds. Each persona has its own name, wake word, voice, and personality â€” so switching personas changes the entire experience.

## What Is a Persona?

A persona is a configuration file ([persona.json](/codex/persona-json)) that tells Beacon:

- **Who** the assistant is (name, tagline, icon)
- **How** it sounds (wake words, TTS voice, STT provider)
- **What** its personality is like (system prompt, tone, traits)

## Customizing Your Persona

### Change the wake word

Edit the `voice.wakeWords` array in your persona's `persona.json`:

```json
{
  "voice": {
    "wakeWords": ["hey jarvis", "yo jarvis"]
  }
}
```

### Change the voice

Pick a different TTS voice or provider:

```json
{
  "voice": {
    "tts": { "provider": "elevenlabs", "voice": "rachel" }
  }
}
```

### Change the personality

Adjust the system prompt, tone, and traits:

```json
{
  "personality": {
    "systemPrompt": "You are a concise, technical assistant...",
    "tone": "professional",
    "verbosity": "concise",
    "traits": ["precise", "efficient"]
  }
}
```

## Creating a New Persona

1. Create a new directory in your personas folder (`~/.beacon/personas/my-persona/`)
2. Add a `persona.json` file following the [persona.json spec](/codex/persona-json)
3. Restart the gateway or reload personas from **Settings â†’ Personas**

Your new persona appears in the persona switcher in the app.

## Installing from Manifold

Browse and install community-created personas from [Manifold](https://manifold.omni.dev):

1. Open **Settings â†’ Personas â†’ Browse Marketplace**
2. Find a persona you like
3. Click **Install**
4. The persona is downloaded to your gateway and ready to use

<Callout type="info">
  Manifold access requires an Omni account. See [Cloud Features](/grid/beacon/cloud-features) for setup.
</Callout>

## Persona Storage

Personas are loaded from these sources, in order of priority:

| Source | Location |
|--------|----------|
| Local | `~/.beacon/personas/` (or custom `BEACON_PERSONAS_DIR`) |
| Manifold | Installed from the persona marketplace |
| Bundled | Built into the gateway binary (Orin) |
