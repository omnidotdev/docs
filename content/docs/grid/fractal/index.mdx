**Omni Fractal** is a fully managed platform for deploying and running your
applications, databases, and services. Push from a Git repository or point Fractal
at a container image, and it builds, ships, and serves your app over HTTPS, with a
managed database, autoscaling, and push-to-deploy built in. Work visually in the
[dashboard](https://fractal.omni.dev) or script everything with the `fractal` CLI.

## Key features

- **Push-to-deploy**: connect a GitHub repository and every push to the tracked
  branch builds and rolls out a new version with zero downtime
- **Deploy from anywhere**: deploy from a Git repository or a prebuilt container
  image
- **One-click templates**: deploy WordPress, n8n, Metabase, Ghost, and more in a
  click &mdash; any companion database is provisioned and wired in for you
- **Managed databases**: high-availability Postgres with automatic scheduled
  backups (configurable schedule and retention) and one-click restore, no
  database to run yourself
- **Private networking**: services in a project reach each other privately at
  `<service>:<port>`, no public exposure required
- **Automatic HTTPS**: certificates are provisioned and renewed for every exposed
  service, including your custom domains
- **Custom domains**: bring your own domain with copy-paste DNS records and
  automatic, zero-downtime certificate issuance
- **Autoscaling and scale-to-zero**: scale on demand, or let idle services sleep
  and wake instantly on the next request
- **Preview environments and canaries**: an ephemeral environment for every pull
  request, plus weighted canary rollouts for safe releases
- **Container shell**: open an interactive terminal in a running container from
  the dashboard to inspect a live service, role-gated and audited
