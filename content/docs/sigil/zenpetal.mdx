**Omni Zenpetal** is the canonical reference implementation of the [ðŸŽ¨ Sigil](/sigil) design system built with [Panda CSS](https://panda-css.com) and [Ark UI](https://ark-ui.com). It provides styled components and design tokens for building modern, accessible interfaces with CSS-in-JS.

## Key Features

- **Design Token Integration**: Full implementation of Sigil's design tokens (colors, typography, spacing, etc.)
- **Styled Components**: Pre-built, accessible React components powered by [Ark UI](https://ark-ui.com) primitives
- **CSS-in-JS**: Type-safe styling with Panda CSS's zero-runtime approach
- **Tree-Shakeable**: Import only the components and styles you need
- **Accessible by Default**: Components built with WCAG accessibility standards in mind
- **Themeable**: Easy theming with design token customization

## Use Cases

- **Application Development**: Build consistent UIs with pre-styled components
- **Design System Adoption**: Use Zenpetal as a starting point for implementing Sigil
- **Rapid Prototyping**: Quickly prototype interfaces with ready-to-use components
- **Custom Theming**: Extend and customize the design tokens for your brand

## How It Works

Zenpetal uses Panda CSS to generate atomic CSS classes from design tokens defined in the Sigil design system, layered on top of Ark UI's accessible component primitives. Components are built with accessibility and consistency in mind, providing a seamless development experience.

```tsx
import { Button, Card, Text } from "@omnidotdev/zenpetal";

function Example() {
  return (
    <Card>
      <Text as="h2">Welcome to Zenpetal</Text>
      <Text>Build beautiful, accessible interfaces with ease.</Text>
      <Button colorPalette="accent">Get Started</Button>
    </Card>
  );
}
```

## Design Tokens

Zenpetal implements the full Sigil design token specification:

- **Colors**: Semantic color scales with light/dark mode support
- **Typography**: Font families, sizes, weights, and line heights
- **Spacing**: Consistent spacing scale for margins and padding
- **Radii**: Border radius values for consistent roundness
- **Shadows**: Elevation system for depth and hierarchy
- **Animations**: Motion tokens for transitions and animations
