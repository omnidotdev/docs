**Omni Charm** is a community-management bot for Discord. It brings the tools a
server needs, welcome cards, moderation, reaction roles, and event logging,
together behind a genuinely polished experience, and it is free and open-source
end to end. The features other bots charge for, like per-server branding, are
included.

## Why Charm?

- **Polished welcome cards**: a rendered, per-member card (avatar, greeting,
  member count) on a themed background, with color emoji support
- **Free theming and branding**: named presets plus full custom colors, and a
  per-server footer and author brand on every embed, never behind a paywall
- **Moderation with receipts**: warn, timeout, kick, ban, and purge, each with an
  auto-numbered case and a themed mod-log
- **Self-service roles**: auto-assign roles on join, and reaction roles members
  claim themselves
- **Event logging**: member joins and leaves, and message deletes and edits, to a
  log channel of your choice
- **Self-hostable**: run your own instance with the official Docker Compose stack

## Architecture

- `charm`: the bot gateway, a Rust service built on Serenity and Poise
- `charm-db`: a Postgres database holding per-guild configuration
- `charm-app` and `charm-api`: the configuration dashboard, planned

Charm connects out to Discord's gateway and needs no inbound network access. All
configuration lives per guild, so one instance can serve many servers, each with
its own theme, channels, and settings.
