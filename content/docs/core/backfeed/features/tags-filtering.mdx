title: Tags & Filtering
description: Organizing the board with tags, filters, search, and saved views
---

Keep a growing board navigable.

## Tags

Project-scoped labels for categorizing posts. Each board defines its own tags;
posts can carry several.

## Filtering, search & sorting

Filter and search the board, and sort it (for example by top-voted or most
recent) to find and prioritize what matters.

## Saved views

Save view configurations (a combination of filters, search, and sort) for quick
recall, so common slices of the board are one click away.
