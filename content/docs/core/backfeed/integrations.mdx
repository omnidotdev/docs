Backfeed integrates across the Omni ecosystem and beyond:

- **Runa** for turning accepted feedback into tracked work (see [Closing the Loop](/core/backfeed/runa-loop))
- **Additional feedback sources and AI triage** (see [AI &amp; Ingestion](/core/backfeed/features/ai-ingestion))
- **GitHub** (planned)

## Ecosystem events

Every change emits a [CloudEvent](https://cloudevents.io) (`backfeed.post.*`,
`backfeed.comment.*`, `backfeed.vote.*`, and more) so the rest of the Omni
ecosystem can react, including [closing the loop with Runa](/core/backfeed/runa-loop).
A documented public API and webhooks are [planned](/core/backfeed/api).
