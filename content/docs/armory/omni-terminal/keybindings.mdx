# Keybindings

Omni Terminal ships with sensible default keybindings for macOS and Linux/Windows. All bindings can be overridden in your [config file](/armory/omni-terminal/configuration#custom-keybindings).

## macOS

### General

| Action | Keys |
|--------|------|
| Copy | `Cmd+C` |
| Paste | `Cmd+V` |
| Clear history | `Cmd+K` |
| New window | `Cmd+N` |
| New tab | `Cmd+T` |
| Close tab/split | `Cmd+W` |
| Open config | `Cmd+,` |
| Quit | `Cmd+Q` |
| Hide | `Cmd+H` |
| Minimize | `Cmd+M` |
| Toggle fullscreen | `Alt+Enter` |

### Font Size

| Action | Keys |
|--------|------|
| Increase | `Cmd+=` |
| Decrease | `Cmd+-` |
| Reset | `Cmd+0` |

### Opacity

| Action | Keys |
|--------|------|
| Increase | `Cmd+Alt+=` |
| Decrease | `Cmd+Alt+-` |
| Reset | `Cmd+Alt+0` |

### Tabs

| Action | Keys |
|--------|------|
| Next tab | `Ctrl+Tab` |
| Previous tab | `Ctrl+Shift+Tab` |
| Previous tab | `Cmd+[` |
| Next tab | `Cmd+]` |
| Tab 1–8 | `Cmd+1` through `Cmd+8` |
| Last tab | `Cmd+9` |

### Splits

| Action | Keys |
|--------|------|
| Split right | `Cmd+D` |
| Split down | `Cmd+Shift+D` |
| Next split | `Cmd+]` |
| Previous split | `Cmd+[` |
| Move divider up | `Cmd+Ctrl+Up` |
| Move divider down | `Cmd+Ctrl+Down` |
| Move divider left | `Cmd+Ctrl+Left` |
| Move divider right | `Cmd+Ctrl+Right` |

### Search

| Action | Keys |
|--------|------|
| Forward search | `Cmd+F` |
| Backward search | `Cmd+B` |

### Scrolling

| Action | Keys |
|--------|------|
| Scroll to top | `Shift+Home` |
| Scroll to bottom | `Shift+End` |
| Page up | `Shift+PageUp` |
| Page down | `Shift+PageDown` |

## Search Mode

These bindings are active while the search bar is open (all platforms):

| Action | Keys |
|--------|------|
| Cancel search | `Ctrl+C` or `Escape` |
| Clear search field | `Ctrl+U` |
| Delete last word | `Ctrl+W` |
| Previous search history | `Ctrl+P` or `Up` |
| Next search history | `Ctrl+N` or `Down` |
| Next match | `Enter` (forward) |
| Previous match | `Enter` (backward) |

## Hints

Hints allow keyboard-driven interaction with URLs and other patterns in the terminal output.

| Action | Keys |
|--------|------|
| Activate hints | `Ctrl+Shift+O` |
| Mouse hints | Hold `Super` (macOS) or `Alt` (Linux/Windows) |

The default hint alphabet is `jfkdls;ahgurieowpq`. When hints are activated, matching text is labeled with characters from this alphabet — type the label to trigger the action (open URL, copy, etc.).
